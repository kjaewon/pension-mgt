# 📋 My Portfolio Rebalancer 기능 명세서 (Feature Specification)

해당 문서는 PRD(개발 요구사항 명세서)를 바탕으로 실제 구현해야 할 구체적인 기능 단위로 세분화한 명세서입니다.

---

## 1. 계좌 관리 기능 (Account Management)

- **기능 설명:** 사용자가 자산을 운용하는 다양한 종류의 계좌를 등록하고 관리하는 기능입니다.
- **세부 기능 목록:**
  1. **계좌 추가 (Create):** 계좌명(별칭)과 계좌 종류(DC, 연금, IRP, ISA, 일반 위탁 등)를 입력하여 새로운 계좌를 생성합니다.
  2. **계좌 목록 조회 (Read):** 등록된 모든 계좌를 목록화하여 보여주며, 글로벌 내비게이션(Tab 또는 드롭다운)을 통해 계좌 간 이동을 지원합니다.
  3. **계좌 정보 수정 (Update):** 생성된 계좌의 이름이나 속성을 변경합니다.
  4. **계좌 삭제 (Delete):** 더 이상 관리하지 않는 계좌를 삭제합니다. (삭제 시 하위 자산 데이터도 함께 처리 또는 경고 팝업 제공)

## 2. 자산 관리 기능 (Asset Management)

- **기능 설명:** 특정 계좌 내에 보유하고 있는 개별 주식, 현금, 채권 등의 자산 데이터를 등록하고 관리합니다.
- **세부 기능 목록:**
  1. **자산 추가 (Create):**
     - **주식/ETF:** 종목명, 종목코드(티커), 보유 수량, 평균 매수가 입력
     - **현금 및 기타:** 상품명(예: 예수금, 예금액), 총 금액 입력
  2. **자산 조회 (Read):** 계좌별 전체 보유 자산 목록을 데이터 그리드(표) 형태로 랜더링합니다.
  3. **자산 수정 (Update):** 매수/매도로 인해 변경된 보유 수량, 평단가 등을 업데이트합니다.
  4. **자산 삭제 (Delete):** 전량 매도한 종목이나 잘못 입력된 자산 항목을 삭제합니다.

## 3. 실시간 데이터 연동 및 자동 평가 기능

- **기능 설명:** 사용자가 입력한 자산 정보(종목코드 등)를 바탕으로 현재 가치를 계산하고 평가 지표를 제공합니다.
- **세부 기능 목록:**
  1. **현재가 연동 연산:** 주식/ETF의 종목코드를 기반으로 외부 주가 API를 호출하여 최신 `현재가`를 반영합니다.
  2. **자동 계산 로직 연동:**
     - **`평가금액`** = 보유 수량 × 현재가 (현금성 자산은 입력된 총 금액 적용)
     - **`수익률(%)`** = ((현재가 - 평단가) / 평단가) × 100
     - **`계좌 내 비중(%)`** = (개별 자산 평가금액 / 계좌 총 평가금액) × 100
  3. **총합 계산:** 활성화된 계좌의 '총 자산(평가금액 합계)', '총 평가손익', '총 수익률'을 산출합니다.

## 4. 목표 비중 기반 리밸런싱 시뮬레이터 (Rebalancing Calculator)

- **기능 설명:** 설정된 목표 비중에 도달하기 위해 매수 또는 매도해야 할 수량/금액을 자동 계산해 주는 핵심 엔진입니다.
- **세부 기능 목록:**
  1. **목표 비중 설정/수정:** 데이터 그리드 내 각 자산 행(Row)에 사용자가 직접 '목표 비중(%)' 값을 입력/수정할 수 있는 입력 필드를 제공합니다.
  2. **비중 100% 검증:** 설정한 목표 비중의 총합이 100%를 초과하거나 미달하는지 검증하고 UI로 피드백을 줍니다. (예: 텍스트를 빨간색으로 표시)
  3. **리밸런싱 액션 자동 산출:**
     - **`목표 금액`** = 전체 계좌 총 자산 × 입력된 목표 비중(%)
     - **`매수/매도 필요액`** = 목표 금액 - 현재 해당 자산의 평가금액
     - **`매수/매도 수량`** = 매수(매도) 필요액 / 현재가 (결과값에서 소수점 올림/버림 로직 적용)
  4. **추천 액션 UI 표시:** 표 내에 배지(Badge)나 색상 텍스트를 이용해 "n주 매수", "n주 매도"와 같은 직관적인 가이드를 표시합니다.

## 5. 대시보드 및 데이터 시각화 (Dashboard & Visualization)

- **기능 설명:** 계좌 상세로 들어가기 전, 전체 로그인 유저의 자산 상태와 목표 비중 현황을 한눈에 파악합니다.
- **세부 기능 목록:**
  1. **통합 요약 카드:** 모든 계좌를 아우르는 '총 자산 규모', '전체 보유 현금액(대기자금)', '통합 수익률'을 요약해 상단에 노출합니다.
  2. **비교 시각화 차트:**
     - 현재 보유 중인 자산군(주식, 채권, 현금)의 실제 비중과 사용자가 이상적으로 설정한 목표 비중 간의 차이를 막대(Bar)나 파이(Pie) 차트로 렌더링합니다.

## 6. 추가 예정/선택형 기능 (Optional Phase)

- **입출금(원금 변동) 기록:** 단순히 계좌에 돈을 넣고 뺀 이력을 관리해 수익률 계산의 오류(착시 효과)를 방지하는 기능.
- **리밸런싱 히스토리 스냅샷:** "2023-11-01 리밸런싱 완료" 와 같이 과거 특정 시점의 포트폴리오 비중과 상태를 기록하고 저장해 두는 기능.
- **반응형 상태 관리(모바일):** PC 뷰뿐만 아니라 모바일 브라우저에서도 그리드(표)가 깨지지 않고 스크롤/스와이프 되도록 모바일 전용 뷰 제공.
